Frequently Asked Questions

What is SSH and how does it work? +
SSH (Secure Shell) is a cryptographic network protocol for operating network services securely over an unsecured network. It provides a secure channel over an unsecured network in a client-server architecture, connecting an SSH client with an SSH server.
Is using SSH legal? +
Yes, using SSH is completely legal in most countries. It's a standard protocol for secure remote access and file transfers. However, what you do with your SSH connection must comply with local laws and regulations.
How often are new SSH servers added? +
We add new servers weekly to ensure we always have fresh, high-quality connections available. Server locations rotate to provide global coverage.
Can I use SSH on my mobile device? +
Absolutely! We provide configuration files compatible with popular SSH clients on both Android and iOS. Our setup guides include mobile-specific instructions.
